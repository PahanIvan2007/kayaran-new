## 2. СУЩНОСТИ (DATA LAYER)

### Люди и роли
- **Пользователь** — физическое лицо в системе
- **Участник** — роль для участия в событиях
- **Инструктор** — ведёт тренировки и маршруты
- **Судья** — фиксирует результаты матчей
- **Волонтёр** — помогает в проведении событий
- **Франчайзи** — владелец точки
- **Администратор** — управляет системой

### География и инфраструктура
- **Точка** — база/клуб/франшиза (физическая локация)
- **Зона активности** — безопасные и опасные зоны
- **Маршрут** — путь движения с уровнями сложности
- **Контрольная точка** — GPS-точка на маршруте

### Оборудование
- **Лодка** — основной объект проката (типизированная)
- **Инвентарь** — вёсла, жилеты, ворота, буи и т.д.

### Активности (ядро)
- **Событие** — универсальный контейнер активности
  - аренда, тренировка, маршрутный выход, матч, турнир, инклюзивная программа

### Спорт
- **Команда** — группа участников
- **Матч** — спортивное соревнование двух команд
- **Турнир** — система матчей с турнирной логикой

### Операции
- **Аренда** — связка пользователь-лодка-событие
- **Платёж** — финансовый след события
- **Журнал** — лог всех действий

### Данные и следы
- **GPS-трек** — последовательность координат
- **Лог события** — системный журнал
- **Отчёт** — агрегированные данные по событию

## 3. СВЯЗИ (RELATIONSHIP LAYER)

```
Пользователь ──участвует──▶ Событие
Инструктор   ──ведёт─────▶ Событие
Судья        ──обслуживает─▶ Матч
Точка        ──проводит──▶ Событие
Событие      ──использует─▶ Маршрут
Событие      ──создаёт───▶ GPS-трек
Пользователь ──арендует──▶ Лодка
Лодка        ──закреплена─▶ Точка
Команда      ──участвует──▶ Матч
Матч         ──часть──────▶ Турнир
Событие      ──формирует──▶ Отчёт
```

## 4. ГЛАВНАЯ АРХИТЕКТУРНАЯ ИДЕЯ

1. **Всё в системе — это Событие** (аренда, спорт, маршрут, тренировка, инклюзия)
2. **Всё связано через GPS + время + точку**
3. **Модули НЕ владеют данными — они их используют**
4. **PWA — только интерфейс, не система хранения**

## 5. ПРИНЦИПЫ ОБМЕНА ДАННЫМИ

| Компонент | С чем общается | Протокол |
|-----------|---------------|----------|
| PWA | API Gateway | HTTPS + WebSocket |

| Судейская панель | API (realtime) | WebSocket |
