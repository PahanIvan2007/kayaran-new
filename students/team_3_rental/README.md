# Команда 3: Прокат

## Домен
Лодки, инвентарь, аренда, QR-идентификация, выдача/возврат, повреждения.

## Таблицы-владельцы
- `boats` — лодки
- `inventory_items` — инвентарь
- `rentals` — аренда (привязана к events)
- `rental_inventory` — инвентарь в аренде

## API
- `GET/POST /boats` — управление лодками
- `GET /boats/available?point_id=` — доступные
- `POST /rentals` — создать аренду
- `PUT /rentals/{id}/return` — возврат
- `PUT /rentals/{id}/damage` — повреждения
- `GET /rentals/user/{id}` — история
